Movie API app

Objective:

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies!!

Movie Endpoints with Documentation

Request URL Response
GET /topmovies Returns a list of all movies.
GET /topmovies/:title Returns data about a single movie.
GET /topmovies/genre/:genreName Returns data about a genre.
GET /topmovies/director/:name Returns data about a director.

User Endpoints

Request URL Response
POST /users Adds a new user.
PUT /users/:id Updates user information.
POST /users/:id/:title Adds a favorite movie to user's list.
DELETE /users/:id/:title Removes a favorite movie from user's list.
DELETE /users/:id Deletes an existing user.
