Movie API app

Objective:

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies!!

Endpoints with Documentation

Description URL Method Type Query Parameters Request Body Format Response Data Format
Add a new user /users POST None { "name": "Ben", "favoriteMovies": ["Mulholland Drive"] } { "name": "Ben", "favoriteMovies": ["Mullholland Drive"] "id": "f34e1bec-325a-45fb-ba2a-81302ea30337" }
Allow users to update their information. /users/:id PUT None { "name": "Ben Phillips" "favoriteMovies": [] } { "id": 1, "name": "Ben Phillips", "favoriteMovies": ["Mulholland Drive"] }
Allow users to add a favorite movie. /users/:id/:title POST None None in the body, title as a parameter "Spirited Away has been added to user 1's array"
Allow users to delete a favorite movie. /users/:id/:title DELETE None None in the body, title as a parameter "Spirited Away has been removed from user 1's array"
Allow existing user to deregister. /users/:id DELETE None None in the body "user 2 has been deleted"
Return a list of all movies. /topmovies GET None None Returns array of movie objects
Return a single movie. /topmovies/:title GET None None Returns details of a single movie
Return a genre. /topmovies/genre/:genreName GET None None Array of movie objects with the specified genre
Return information on director. /topmovies/director/:name GET None None Returns details of a movie directed by a specific director
